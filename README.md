## 1. 서비스 소개

   ### 서비스 설명:


    1. 직접 사용자의 손에 네일 디자인을 입혀볼 수 있는 서비스 제공
    2. 디자인을 입혀보고 바로 네일샵까지 예약을 할 수 있는 기능 제공
    3. 사용자가 각 네일샵의 별점과 리뷰를 볼 수 있음

# 2. 주요 기능

고객 - 
이미지 오버레이를 통한 네일에 디자인 입히기 기능
원하는 가게의 위치 확인가능
원하는 날짜와 시간에 서비스 예약 가능
가게의 리뷰를 확인가능


가게 관리자 - 
가게 등록 후 썸네일 이미지 업로드 가능
가게의 디자이너 추가 및 삭제 가능
가게의 스케줄 관리 가능
