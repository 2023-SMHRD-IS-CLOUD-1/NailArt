NailArt(네일아트) 팀명: 손톱뜯지마

1. 서비스 소개
● 서비스 설명:

  ○ 직접 사용자의 손에 네일 디자인을 입혀볼 수 있는 서비스 제공
  ○ 디자인을 입혀보고 바로 네일샵까지 예약을 할 수 있는 기능 제공
  ○ 사용자가 각 네일샵의 별점과 리뷰를 볼 수 있음
